Subject Build Configurator Maven Plugin and Extension 1.0.0 Released
Date Thu, 30 Aug 2012 16:14:21 GMT 
Hello everyone,

I recently released Build Configurator Maven Plugin and Build Configurator
Maven Extension, version 1.0.0.

With Build Configurator you can:
* Load multiple property files.
* Add or update dependencies and dependency exclusions.
* Add or update plugins, plugin executions, and plugin dependencies.
* Use Expression Language in property files and Build Configurator
configuration.

Please take a look and tell me what you think.

Project site: http://htfv.github.com/build-configurator/
Sources: https://github.com/htfv/build-configurator
Issues: https://github.com/htfv/build-configurator/issues
