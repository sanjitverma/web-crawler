I use this info sometimes to find the right groupId and artifactId. m2e
also uses this info to download sources automatically, and I use this
feature very often. You can remove this files by setting addMavenDescriptor
to false in archiver configuration [1], but I wouldn't do that.

[1] http://maven.apache.org/shared/maven-archiver/index.html
