Hi David, 

It looks like you have an unsigned SSL certificate on your SVN server.  So you need to install
the certificate in the Java keystore of hosts that need to run the release plugin.  There
are a lot of examples of this on the net already, just google for "install unsigned cert java"
or whatnot.

Cheers, Brian
