This is more of a Java question than a Maven question at this point.  Maven uses Java to validate
certificates as a part of channel security.  If you don't want channel security, don't use
SSL ;)

I don't have enough Windoze foo to help you past this point, maybe someone else on the list
does.  But you shouldn't have had these problems if you read those google results closely
enough and I bet you can solve it yourself before someone else were to respond...

Good luck!
