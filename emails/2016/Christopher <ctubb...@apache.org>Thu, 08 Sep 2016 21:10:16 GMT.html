If it helps anybody else, I ended up using Guava's
ClassInfo.getResourceName() and concatenated that with
project.getBuild().get[Test]OutputDirectory() and checked that with
File.exists(). I then encapsulated that in a Java 8 Predicate called
"isProjectClass()" and used that as a Java 8 Stream filter. I'll soon be
pushing this to https://github.com/revelc/warbucks-maven-plugin, if anybody
is interested in reusing my strategy.
